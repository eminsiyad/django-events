# Django Events Web Application

A full-featured Django web application for managing and displaying events.

##  Live Demo
https://django-events.onrender.com

---

##  Features

- User authentication (Login / Register / Logout)
- Event creation and management
- Admin panel
- Template-based frontend
- Static file handling
- Production deployment using Gunicorn
- Hosted on Render

---

##  Tech Stack

- Python 
- Django
- SQLite (default database)
- Gunicorn
- HTML / CSS
- Render (Deployment)
- GitHub (Version Control)
